# 🖤 Mr. Chao Tattoo Bot - Telegram Mini App

Мини-приложение для записи на тату-сеансы через Telegram.

## 🚀 Деплой на Vercel

1. Fork этот репозиторий на GitHub
2. Зарегистрируйся на [Vercel](https://vercel.com)
3. Нажми "Import Project" и выбери этот репозиторий
4. Vercel автоматически определит настройки
5. Нажми "Deploy"

## 🔗 Подключение к Telegram боту

1. Получи URL после деплоя (например: `https://tattoo-bot.vercel.app`)
2. Напиши @BotFather в Telegram
3. Выбери `/mybots` → `@Mr_Chao_tattoo_bot`
4. `Bot Settings` → `Menu Button` → `Edit URL`
5. Вставь URL от Vercel
6. Готово! 🎉
